# 🎨 AI Gesture Air Canvas Pro

## 🚀 Overview
AI Gesture Air Canvas Pro is a computer vision-based system that allows users to draw in the air using hand gestures without touching any device.

---

## ✨ Features
- ✋ Hand gesture-based drawing
- 🎨 Real-time air canvas
- 🔷 Shape recognition (circle, rectangle, etc.)
- 🤖 Object detection (car, bicycle)
- 💾 Save drawings

---

## 🧠 Technologies Used
- Python
- OpenCV
- MediaPipe
- Deep Learning (MobileNet SSD)
- NumPy

---

## ⚙️ How It Works
1. Webcam captures real-time video  
2. MediaPipe detects hand landmarks  
3. Gestures control drawing and UI  
4. Shape detection identifies drawn figures  
5. Object detection identifies real-world objects
